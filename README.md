# IBD_Kmer_Analysis

```perl GetNucFrequency_PerSeq_varK.pl ./GCF_000403175.fna 4 > GCF_000403175.4mers```

and then you can count the bias:

```perl kmerbias.pl GCF_000403175.4mers "GGCC,GGCG,GGGC"```
